# v1.0.1

Improved printable kit package for the HO Union Pacific #80 coal turbine.

## Added
- One shell STL per unit.
- Split chassis parts for all three units.
- Chassis part index and updated build guide.
- 3MF print-set previews, faceted STEP references, DXF, SVG, BOM, paint/decal plan, research notes, manifest, and validation report.

## Known Limits
- STEP files are faceted interchange exports generated without a local CAD kernel.
- Donor drive mounting should be adjusted from measured parts before final metal cutting.
- Exterior accuracy is still limited by public/reference-model data until v1.1 CAD and printer feedback are available.
