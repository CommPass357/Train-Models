# HO Union Pacific #80 Coal Turbine v1.0.0

This package is a complete digital manufacturing set for a three-unit HO scale Union Pacific #80 coal turbine: A/control unit, center turbine unit, and Challenger-style coal tender. Each unit has its own shell and chassis file set.

Scale target: 1:87.1. The reported 226 ft prototype length scales to 790.9 mm / 31.14 in overall.

## Included
- Parametric Python source and JSON dimensions in `source/`.
- High-detail resin STL files in `exports/stl/resin/`.
- FDM/plastic STL and 3MF files in `exports/stl/fdm/` and `exports/3mf/`.
- Faceted STEP interchange files and metal plate DXF in `exports/step/` and `exports/dxf/`.
- Paint masks, decal placement guides, glazing guides, BOM, research notes, and build notes in `docs/`.

## Important Limits
These files are a detailed digital model package generated from public dimensional references and photo/model-informed proportions. Final fit depends on the donor trucks, motors, couplers, printer, resin/plastic shrink, and the builder's chosen wiring. The chassis are functional HO designs, not a historical reproduction of the prototype underframe mechanics.
