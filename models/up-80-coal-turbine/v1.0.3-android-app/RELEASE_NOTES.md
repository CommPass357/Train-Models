# v1.0.3 Android App

Native Android catalog app source for the UP #80 model kit.

## Added
- Android app project with Java source.
- Local lightweight preview catalog generated from v1.0.1 STL files.
- Three-shell default preview.
- Full catalog preview.
- GitHub links for each part, each batch, chassis docs, and the full v1.0.1 manufacturing zip.

## Notes
- This is an Android app source release; no full STL files are bundled in the APK assets.
- Printable files remain in v1.0.1.
- APK building requires Android SDK/Gradle or the included GitHub Actions workflow.
